# Nutanix NCM Self Service Demo Blueprints

Welcome to my collection of Nutanix NCM Self Service blueprints.

This repository contains blueprints that I use for demonstrations, testing, learning, and experimentation. Where required, you'll also find supporting content such as scripts, documentation, images, and other assets needed by individual blueprints.

## What to Expect

These blueprints are intended as examples and starting points. They are **not production-ready** and should be treated accordingly.

You should expect that:

* Blueprints are primarily built for demos and lab environments.
* They may contain assumptions about the target environment.
* They may require modification before they work in your environment.
* They may change at any time as I improve, update, or repurpose them.
* Documentation may be minimal, especially for work-in-progress examples.

## Before You Deploy

Please review each blueprint before using it.

In particular, verify:

* Credentials and authentication methods
* Network configuration
* Variables and runtime inputs
* Images and supporting files
* Security settings
* Environment-specific dependencies

Never assume a blueprint is suitable for production without performing your own validation and testing.
